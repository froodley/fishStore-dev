Technologies

PHP is from-scratch REST-ful, "invisible" single-URL MVC architecture with DB-dynamic, encrypted, YAML-driven entity framework.  YAML parser is from-scratch.

Front end uses generated HTML, Angular (also using DB-dynamic entities) for data binding, jQuery and jQuery UI, and vector FontAwesome text icons for minimal graphic downloads
